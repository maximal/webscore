# WebScore

Online music sheets player and catalog for [MuseScore](https://musescore.org/) files.

## Demo

Visit [webscore.maximals.ru](https://webscore.maximals.ru) and try to upload a MSCZ file.

## Stack

* PHP 8.5
* TypeScript
* [Laravel](https://laravel.com/) + [FrankenPHP](https://frankenphp.dev/) + [Inertia.js](https://inertiajs.com/)
* [Vue.js](https://vuejs.org/)
* [Docker](https://www.docker.com/)

### Tools

* [MuseScore](https://musescore.org/) editor
* [Bun](https://bun.sh/) JavaScript package manager
* [Oxc](https://oxc.rs/) JavaScript toolchain
* [Vite](https://vite.dev/) bundler
* [Mago](https://mago.carthage.software/) PHP toolchain
